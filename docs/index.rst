Welcome to ZenIRCBot API's documentation!
=========================================

This is the API for ZenIRCBot_. 

So far it is the `JavaScript package`_ and `Python package`_. You can install it
using::

   $ npm install zenircbot-api

or::

   $ pip install zenircbot_api

.. _`JavaScript package`: http://search.npmjs.org/#/zenircbot-api
.. _`Python package`: https://crate.io/packages/zenircbot_api/
